# SMART-CURRENT-THEFT-DETECTION-WITH-CT-SENSORS-AND-EDGE-AI-
Smart electricity theft detection system using CT and ACS712 sensors with an STM32 microcontroller and Edge AI. It monitors real-time current, detects anomalies indicating illegal usage, and enables fast, low-cost detection. Designed for smart grids, it improves efficiency, reduces losses, and supports future IoT integration.
